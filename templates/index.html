Добрый день. Это краткое руководство для API.

С помощью API Вы можете узнать:

1. Регистрировался ли кто-то с интересующего Вас email адреса на сайте ask.fm

2. Существует ли аккаунт с интересующим Вас именем сайте ask.fm

Для этого используйте ссылку вида:

1. http://host_name/askAPI/find_info/<email>

method: GET, response format: JSON

Используйте интересующий Вас email или аккаунт для получения информации.
